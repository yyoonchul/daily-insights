# 출력 마크다운 템플릿

아래 구조를 따라 정리본을 생성한다.

## 구조

```
# 📰 Daily Digest — YYYY-MM-DD

> N건 정리 | 카테고리1, 카테고리2, ...

---

## 📋 간단 요약

### {글 제목 1} 
**출처:** {출처} · **카테고리:** {카테고리} · **링크:** [원문]({URL})
- 불렛 1
- 불렛 2
- 불렛 3

### {글 제목 2}
**출처:** {출처} · **카테고리:** {카테고리} · **링크:** [원문]({URL})
- 불렛 1
- 불렛 2
- 불렛 3

(... 모든 글을 같은 형식으로 나열)

---

## 📝 상세 정리

### 1. {글 제목 1}


{상세 정리 본문 20~30줄}

---

### 2. {글 제목 2}


{상세 정리 본문 20~30줄}

(... 간단 요약과 같은 순서로 나열)
```

## 규칙

- 간단 요약과 상세 정리의 글 순서는 동일해야 한다.
- 상세 정리 사이에 `---` 구분선을 넣는다.
- 메모에 "요약만"이 있는 글은 상세 정리 섹션에서 제외한다.
- 수집 실패한 글은 간단 요약에 `⚠️ 수집 실패`로 표기하고 상세 정리에서 제외한다.
